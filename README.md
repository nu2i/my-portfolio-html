# Personal Portfolio — Wejdan Dakhel

Single-page personal portfolio built with plain HTML, CSS, and JavaScript.
No frameworks — just a design that reflects who I am: an Information Technology
student at Taif University with a focus on cybersecurity and ethical hacking.

## Design concept

The page opens with a simulated terminal window that "types" out basic info
(name, degree, university) — a nod to the command-line world of cybersecurity,
paired with a dark, monospace-heavy theme and a small "connection encrypted"
status indicator for extra flavor.

## Tech used

- HTML5
- CSS3 (custom properties, animations, responsive layout)
- Vanilla JavaScript (typing effect, scroll reveal)
- Google Fonts: Fira Code, IBM Plex Mono
